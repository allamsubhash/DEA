# Smart Support AI - Customer Support Ticket Classification and Big Data Analytics Platform

Smart Support AI is a production-ready, full-stack platform designed for automated customer support ticket classification and deep analytics using Big Data and Machine Learning.

## 🚀 Features

- **Comprehensive Coverage**: Specialized pages for managing tickets, agents, models, and data pipelines.
- **AI-Powered Classification**: Automated ticket categorization and urgency detection using NLP (TF-IDF + Logistic Regression).
- **Sentiment Analysis**: Layered rule-based engine for real-time customer sentiment tracking.
- **Modern SaaS UX**: Premium dashboard experience built with React 19, Tailwind CSS v4, and Framer Motion.
- **Big Data Simulation**: Pipeline monitoring and real-time streaming visibility.
- **Advanced Visualizations**: Interactive charts for trends, root cause analysis, and forecasting.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS v4, Framer Motion, Recharts.
- **Backend**: FastAPI, Scikit-Learn, Pandas, NumPy.
- **Database**: Mock DB (Session-based) for high-speed simulation.

## ⚙️ Installation & Setup

### Backend
1. Navigate to `backend/`
2. Create virtual environment: `python3 -m venv venv`
3. Activate venv: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run the server: `uvicorn main:app --reload`

### Frontend
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
